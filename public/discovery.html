WHAT IS STARCOIN

Starcoin provides the utmost security from the origin based on enhanced PoW consensus and Secure smart contract, using the language of Move. Through layered and flexible interoperability, it optimizes the building of ecosystems such as DeFi, NFTs, Gaming, etc., with higher efficiency and convenience. This process redefines value by empowering every participant in the ecosystems to enjoy the multiplication of values.

STARCOIN

Starcoin combines the benefits of secure smart contracts and a layered network to redefine the value of Web 3.0 by ensuring utmost security of data on the chain.

About

move

Using Move, a smart contract language for resource programming, Starcoin has developed secure digital asset protocols including FT and NFT, providing formalized verification tools to replace traditional contract auditing and further reinforce the security of contracts and the security of user assets on the chain.

Starcoin

LAYER 1

Starcoin's Layer 1 is a decentralized, open network of smart contracts that everyone can participate in, giving users the ownership of their own data, while being fully enabled to sense network congestion in real time and dynamically adjust network-wide configurations to improve network utilization.

Starcoin

LAYER 2

Starcoin's Layer 2 is an efficient, scalable application network capable of secure and seamless states transfer, easy and fast data scaling, low latency, and great operability.
