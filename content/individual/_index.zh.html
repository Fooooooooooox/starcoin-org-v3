Another more geeky way to get started with Starcoin is by using Starcoin nodes. starcoin nodes have all the data, so owning a node enables you to accomplish many things. These include common operations, such as starting a poll, Mint STC, viewing the status of all accounts. It is recommended for geeks to build their own Starcoin node. This module contains the instructions for installing, initiating, and using the nodes.
